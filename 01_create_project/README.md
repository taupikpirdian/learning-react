
# Installation React + Vite

for create a project, we can use two ways:

## Installation

Install my-project with npm, before we can go into the project folder:
```bash
  cd project_folder
```
Installation command for Javascript (jsx):

```bash
  npm create vite@latest . -- --template react
```

Installation command for Typescript (tsx):
```bash
  npm create vite@latest . -- --template react-ts
```
