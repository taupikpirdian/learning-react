# Learning React

Repository ini berisi kumpulan project React untuk belajar konsep-konsep dasar React secara bertahap. Setiap folder merepresentasikan satu topik pembelajaran dengan contoh implementasi yang praktis.

## Tujuan

Mendokumentasikan proses belajar React dengan membuat beberapa folder project React, di mana setiap folder fokus pada satu konsep atau topik tertentu.

## Daftar Materi

Berikut adalah urutan pembelajaran yang direkomendasikan:

### 01. [Create Project](./01_create_project/)
- Setup project React dengan Vite
- Konfigurasi dasar React
- Memahami struktur project React

### 02. [Hello World](./02_hello_world/)
- Membuat component pertama
- Memahami JSX
- Cara merender component ke browser

### 03. [Props](./03_props/)
- Mengirim data antar component
- Destructuring props
- Membuat component reusable

### 04. [Nested Component](./04_nested_component/)
- Component di dalam component
- Hierarki component
- Komposisi component

### 05. [Style Component](./05_style/)
- Styling component dengan CSS
- Penggunaan className di React
- Import file CSS

### 06. [Conditional Rendering](./06_condition/)
- Conditional dengan if statement
- Ternary operator
- Logical AND (&&)
- Null component

## Cara Menggunakan Repository Ini

1. **Clone repository**
   ```bash
   git clone <repository-url>
   cd learning-react
   ```

2. **Pilih topik yang ingin dipelajari**
   - Setiap folder bisa berdiri sendiri sebagai project React
   - Disarankan mengikuti urutan nomor dari 01 ke 06

3. **Jalankan project tertentu**
   ```bash
   cd 02_hello_world  # atau topik lain
   npm install
   npm run dev
   ```

4. **Baca README di setiap folder**
   - Setiap topik memiliki README sendiri dengan penjelasan lengkap
   - Ada contoh kode dan implementasi

## Tech Stack

- **React** - JavaScript library untuk building UI
- **Vite** - Build tool yang cepat untuk modern web apps
- **JavaScript (ES6+)** - Bahasa pemrograman yang digunakan

## Konsep yang Dipelajari

- Component-based architecture
- JSX syntax
- Props dan state
- Conditional rendering
- Styling di React
- Nested dan komposisi component

## Cara Belajar

1. Baca README di setiap folder untuk memahami konsep
2. Lihat implementasi kode di folder `src`
3. Jalankan dan eksperimen dengan kode
4. Modifikasi dan coba variasi baru
5. Lanjut ke topik berikutnya
